# PolancoWatch Frontend

React + TypeScript + Vite

## Overview
PolancoWatch frontend is a modern single‑page application that visualizes real‑time system metrics streamed from the backend via SignalR. It uses Tailwind CSS with a glassmorphism design, Recharts for charts, and Lucide icons.

## Prerequisites
- Node.js >= 18
- npm (or yarn)
- Docker (optional for production)

## Getting Started
```bash
# Clone the repo
git clone https://github.com/yourusername/PolancoWatch.git
cd PolancoWatch/PolancoWatch_frontend

# Install dependencies
npm install --legacy-peer-deps   # required for older Vite/Tailwind versions
# or: npm ci

# Development server
npm run dev
```
Open `http://localhost:5173` (or the URL shown in the console).

## Build for Production
```bash
# Set the API URL that the frontend will call (replace with your domain)
docker build --build-arg VITE_API_URL=http://your-domain.com -t polancowatch-frontend .
# Or, if using Dokploy, set the build‑time argument `VITE_API_URL` in the UI.
```
The built assets are served by Nginx in the Docker image.

## Environment Variables
| Variable | Description | Example |
|----------|-------------|---------|
| `VITE_API_URL` | Base URL of the backend API (must be reachable from the browser) | `http://myserver.com` |
These variables are injected at **build time**; they are not read at runtime.

## Deployment with Dokploy
1. Create a **Compose** application.
2. Add the `docker-compose.yml` from the backend repository (which also builds the frontend).
3. In the **Frontend** service, set the build‑time argument `VITE_API_URL` to the domain generated by Dokploy.
4. Deploy.
